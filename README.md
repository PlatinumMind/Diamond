### 🏠 [Homepage](https://github.com/PlatinumMind/Diamond)

## Install

```sh
# linux install script
wget -qO- https://raw.githubusercontent.com/PlatinumMind/Diamond/main/install.sh | bash
```

## Usage

```
just run dbuild in a directory with a diamond.build
```
